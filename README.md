# Excel Capstone Project – E-Commerce Sales Analysis

## Project Overview
This project analyzes an e-commerce sales dataset using Microsoft Excel. 
The objective is to perform data cleaning, transformation, exploration, 
and analysis to identify business insights.

## Tools Used
- Microsoft Excel

## Project Tasks
- Data Cleaning
- Data Transformation
- Data Exploration
- Sales Analysis

## Key Insights
- Certain product categories generate higher sales revenue compared to others
- Customer loyalty level influences purchasing frequency
- Some regions and stores outperform others in terms of sales volume
- Payment methods show varying popularity among customers.

## Conclusion
The analysis of the E‑Commerce sales dataset demonstrates how Excel can be used to transform raw data into meaningful business insights. Through data cleaning, transformation, and analysis, important patterns related to sales performance, customers, and products were identified. These insights can help businesses improve decision‑making and optimize operations.
